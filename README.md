**Expense Tracker – CLI Application (Java)**

**Overview**

The Expense Tracker is a command-line based Java application designed to help users efficiently manage their daily expenses.
It enables users to set a total budget, record expenses, monitor total spending, and track remaining balance in real time.
This project demonstrates strong fundamentals in Java, Object-Oriented Programming (OOP), and menu-driven application design.

**Key Features**

Budget initialization at program start
Add new expense entries
View all recorded expenses
Calculate total spending
Display remaining balance
Budget exceed warning system
Delete specific expense entries
Clean and user-friendly CLI interface

**System Design**

The application follows a modular structure:
Expense Class
Encapsulates expense data (name and amount).
ExpenseTracker Class
Manages application logic including:
Expense storage using ArrayList
Menu handling
Budget tracking
CRUD operations
This separation ensures better maintainability and scalability.
**Technologies Used**

Java
ArrayList (Dynamic Data Storage)
Scanner (User Input Handling)
Object-Oriented Programming Principles
Git & GitHub for Version Control
